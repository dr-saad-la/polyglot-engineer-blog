## Local Development

```bash
# Prerequisites: UV, just, Python 3.12+

# Clone and setup
git clone https://github.com/dr-saad-la/polyglot-engineer-blog.git
cd polyglot-engineer-blog
just install

# Serve locally
just serve
```

Visit: http://127.0.0.1:8000/polyglot-engineer-blog/
